To run this simulation, the folder in this location must be selected in the Unity Interface and loaded as
a Unity project.

Doing this will allow you to run the simulation as soon as it opens.

The test environment is called SwarmTopologyTestEnvironment and should be the first scene to start up when
the project is opened.

To switch topologies, select each drone to eneble or disable whichever topology script/s you wish to run.
